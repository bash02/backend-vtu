# Bun Support

This project is Bun-ready!

## Install dependencies

```bash
bun install
```

## Run in development mode

```bash
bun run dev
```

## Run in production mode

```bash
bun run start
```

## Notes
- Bun uses `bun.lockb` (or `bun.lock`) for dependency locking.
- All scripts in package.json are compatible with Bun.
- For more, see https://bun.sh/docs/cli/run
